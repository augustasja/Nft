# Nft

**Installation**
 - Clone repository
 - cd into Nft-App folder
 - rename ```.env.example``` to ```.env```
 - setup database connection
 - run ```composer install```
 - run ```npm install```
 - run ```php artisan key:generate```
 - run ```php artisan migrate```
 - run command ```php artisan api:fetch```
 - run ```php artisan serve```
